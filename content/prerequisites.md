# Prerequisites

Please follow the prerequisites before you begin building the workflow.

## Accessing Amazon Quick Suite

Please use the same AWS environment on "A Complete Guide to Amazon Quick Suite" workshop to continue the buiding the automate workflow. 

So now, you have access to the AWS Console, follow these steps to access Amazon Quick Suite:
- **Verify your region:** Before proceeding, check that you're in the correct AWS region by looking at the region selector in the top-right corner of the AWS Console. You can identity the appropriate region from the workshop studio event under the **AWS account access** section.

This section will guide you through the necessary setup steps to prepare your environment for the workshop.

![Open AWS Console](images/open-aws-console.png)

- **Navigate to Amazon Quick Suite:** In the search bar at the top, type `Quick Suite` and select Amazon Quick Suite from the services list.
- **Sign in:** If you are prompted to sign in. Enter the following email: `workshop-user@example.com`
- **Verify login** To verify you are logged into Amazon Quick Suite as the correct participant, click the user icon in the top right of the screen, and ensure you are logged in with the following username `WSParticipantRole/Participant`
